A function $ I[A] $ of a set $ A $ defined by $$I[A](x) =  
\left \{ \begin{array}{cc} 
  1 & x \in A \\ 
  0 &  \qquad\mbox{  otherwise  } 
\end{array} \right.$$ There are certain useful identities associated
with these, such as
$$1[A \cap B](x)= min(1[A](x),1[B](x)) = 1[A](x).1[B](x),$$
$$1[A \cup B](x)= max(1[A](x),1[B](x)).$$
