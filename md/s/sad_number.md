A number which is not happy. A number which does not reduce to 1, when
the digits of the number are repeatedly squared and added: e.g.
$36 \rightarrow 3^2 + 6^2 = 45 \rightarrow 4^2 + 5^2 = 41
  \rightarrow 4^2 + 1^2 = 18 \rightarrow 1^2 + 8^2 = 65 \rightarrow 61
  \rightarrow 37\rightarrow 58 \rightarrow 89 \rightarrow 154
  \rightarrow 42 \rightarrow 20 \rightarrow 4 \rightarrow 16
  \rightarrow 37 \rightarrow 58 \rightarrow$ and so on
