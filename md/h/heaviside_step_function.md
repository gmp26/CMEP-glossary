The step function $\mathrm{H}(t)$, defined as: $$\begin{array}{c} 
  \mathrm{H}(t) = 0, \\
  \mathrm{H}(t) = 1, 
\end{array}
\left \{ 
\begin{array}{c} 
  t < 0 \\
  tt \geq 0 
\end{array}
\right .$$

We have $ \mathrm{H}^{\prime} (x)= \delta(x) $ where $\delta$ is the
Dirac delta function; $\mathrm{H}^{\prime}(x)$ is undefined when $x=0$.
