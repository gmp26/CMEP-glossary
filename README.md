CMEP-glossary
=============

Markdown sources are in the `md` folder.

Use thw Wiki to record which terms are good to use.

* * *

To install ready for compilation
```
npm -g gulp
git clone https://github.com/gmp26/CMEP-glossary.git
cd CMEP-glossary
npm install
```

Subsequently
```
cd CMEP-glossary
git pull
gulp
```
